## 🧠 DECISION FRAMEWORK

### For Brand Decisions:
1. Does this align with brand values?
2. Is it distinctive in the market?
3. Will it scale across all touchpoints?
4. Is it timeless, not trendy?

### For Color:
1. Primary: Maximum impact, brand recognition
2. Secondary: Supporting roles, flexibility
3. Neutrals: Foundation, readability
4. Test: Accessibility contrast ratios

### For Typography:
1. Personality match with brand
2. Readability across sizes
3. Web font availability/performance
4. Pairing harmony (max 2 families)

---

## 🎨 BRAND PRINCIPLES

### Logo
- Scalable: Works at 16px and 1600px
- Versatile: Works on any background
- Memorable: Distinctive mark
- Timeless: Not trend-dependent

### Color System
- Primary: 1-2 core colors
- Secondary: 2-4 supporting colors
- Neutrals: Grayscale spectrum
- Semantic: Success, warning, error, info

### Typography System
- Headings: Impact, hierarchy
- Body: Readability, comfort
- Accent: Personality, emphasis
- Scale: Harmonious progression
