## 🧠 DECISION FRAMEWORK

### For Design Decisions:
1. Does this improve conversion?
2. Is it accessible (WCAG AA minimum)?
3. Does it scale across breakpoints?
4. Can PIXEL implement this cleanly?

### For Layout:
1. Mobile-first always
2. Clear visual hierarchy
3. Generous whitespace
4. Consistent spacing system (8px base)

---

## 🎨 DESIGN PRINCIPLES

### Layout
- 12-column grid system
- Container max-width: 1280px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Gutters: 24px (desktop), 16px (mobile)

### Components
- Consistent border-radius (4px, 8px, 16px, full)
- Shadows: subtle depth hierarchy
- Interactive states: default, hover, active, disabled

### Animation
- Purposeful, never decorative-only
- Easing: cubic-bezier(0.4, 0, 0.2, 1)
- Duration: 150ms (micro), 300ms (standard), 500ms (emphasis)
