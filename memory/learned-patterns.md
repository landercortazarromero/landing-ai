## 🎨 DESIGN PATTERNS

### Landing Pages de Alta Conversión
- Hero con CTA claro arriba del fold
- Social proof (logos, testimonios) en los primeros 3 segundos
- Un solo CTA primario por sección
- Mobile-first siempre

### Brand Guidelines Estructura
1. Logo (uso correcto/incorrecto)
2. Paleta de colores (primaria, secundaria, neutros)
3. Tipografía (hierarchy, sizes, weights)
4. Imagery (estilo fotográfico, ilustraciones)
5. Voice & Tone
6. Aplicaciones (web, print, social)

## ⚙️ TECH PATTERNS

### Stack Preferido Web
- Framework: Next.js 14+ (App Router)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Icons: Lucide React
- Deploy: Vercel

### Estructura Componentes
```
/components
  /ui          ← Componentes base (Button, Card, Input)
  /sections    ← Secciones de página (Hero, Features, Footer)
  /layout      ← Layouts (Header, Sidebar, Container)
```

## 🧠 WORKFLOW PATTERNS

### Entrega de Diseño
1. Moodboard + referencias
2. Wireframe (baja fidelidad)
3. Mockup (alta fidelidad)
4. Prototype interactivo
5. Handoff con especificaciones

### Feedback Loop
- Siempre mostrar A vs B, nunca preguntar "¿te gusta?"
- Documentar por qué se rechazó cada opción
- Una decisión = una línea en DECISIONS-LOG.md
