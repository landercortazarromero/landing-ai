# XURL — Elite Twitter Intelligence

## Quick Commands

### Profile Analysis
```bash
xurl analyze @username
xurl profile @username --followers --engagement
```

### Thread Research
```bash
xurl thread THREAD_ID
xurl thread --summarize
xurl thread --export
```

### Content Intelligence
```bash
xurl trending --niche "wordpress|ecommerce"
xurl hashtags --top 20
xurl threads --week --engagement
```

### Lead Generation
```bash
xurl leads --query "need developer" --niche wordpress
xurl leads --query "shopify expert needed"
xurl prospects --min-followers 1000 --max-followers 50000
```

## Usage

### Competitive Analysis
```bash
xurl analyze @competitor --tweets 100
```

### Content Ideas
```bash
xurl trending --niche "motorsport|racing"
xurl threads --viral --topic "racing community"
```

### Outreach
```bash
xurl leads --query "looking for developer"
xurl prospects --location "Spain" --niche "ecommerce"
```
