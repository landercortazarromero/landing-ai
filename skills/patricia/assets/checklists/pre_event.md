# PATRICIA — Checklist Pre-Evento

## 7 días antes
- [ ] Confirmar asistencia mínima (quórum)
- [ ] Reservar casa / lugar
- [ ] Planificar menú comida
- [ ] Preparar lista de material
- [ ] Enviar recordatorio grupo

## 3 días antes
- [ ] Confirmar profesionales invitados
- [ ] Verificar transporte
- [ ] Comprar provisiones base
- [ ] Preparar botiquín
- [ ] Confirmar weather forecast

## 1 día antes
- [ ] Última confirmación participantes
- [ ] Revisar checklist de material
- [ ] Preparar cronograma impreso
- [ ] Chargear baterías de cámaras
- [ ] Preparar contacto de emergencia

## Día del evento
- [ ] Salir a tiempo
- [ ] Llevar cronograma
- [ ] Llevar material deportivo
- [ ] Llevar kit fotografía/documentación
- [ ] Llevar botiquín

## Después
- [ ] Recoger feedback del grupo
- [ ] Publicar contenido (fotos/vídeo)
- [ ] Archivar documentación
- [ ] Planificar siguiente evento
